## Research question
After ChatGPT’s public launch, did software ecosystems that relied more heavily on Stack Overflow experience a larger increase in GitHub issue burden?

## Core variables
- **Outcome variable:** monthly GitHub issue burden, measured with issue-open volume, open-duration, and time-to-close metrics for selected public repositories
- **Treatment variable:** post-ChatGPT exposure interacted with pre-existing Stack Overflow dependence at the ecosystem or repository group level

## Data type
Public Real

## Data note
This project uses public data from Stack Overflow / Stack Exchange and GitHub’s public event history. The goal is to study whether support activity shifted away from public Q&A forums and toward open-source project issue trackers after the arrival of high-usage generative AI tools.
